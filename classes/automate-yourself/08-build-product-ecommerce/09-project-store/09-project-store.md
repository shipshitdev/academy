# Project 2: Build Your First Online Store (Shopify)

## Learning Objective
Hands-on project: Create a complete, professional Shopify store from scratch, applying all setup knowledge you've learned.

## Main Content

Time to build your store. Follow these steps:

**Step 1: Sign Up and Basic Setup**

- Create Shopify account (start with free trial)
- Choose store name (match your niche)
- Complete basic store settings (currency, timezone, address)
- Add contact email for customer support

**Step 2: Choose and Customize Theme**

- Browse themes in Shopify Theme Store
- Choose one that fits your niche (modern, clean, mobile-friendly)
- Install and customize:
  - Upload your logo
  - Set brand colors (use tools like Coolors.co)
  - Choose fonts that match your brand
  - Customize homepage sections

**Step 3: Install Essential Apps**

- Dropshipping app (DSers, Spocket, Zendrop, CJdropshipping, or AutoDS)
- Email marketing (Klaviyo or Omnisend)
- Product reviews (Loox or Judge.me)
- Google Analytics and Meta Pixel

**Step 4: Create Legal Pages**

Create these pages (use templates or generators):
- Privacy Policy
- Terms of Service
- Refund Policy
- Shipping Policy
- About Us (builds trust)

Add them to your footer menu.

**Step 5: Set Up Payments**

- Enable Shopify Payments (if available)
- Add PayPal
- Configure other payment methods
- Test payment processing

**Step 6: Configure Shipping**

- Set shipping zones (start with your target market)
- Create shipping rates:
  - Free shipping (if you'll offer it)
  - Flat rate options
  - Express shipping option
- Set processing times (account for supplier delays)

**Step 7: Import Products**

- Use your dropshipping app to find products
- Import 5-10 products to start
- Edit each product:
  - Write compelling titles
  - Add detailed descriptions
  - Upload high-quality images
  - Set competitive prices
  - Add variants if applicable

**Step 8: Create Collections**

Organize products into collections:
- "Best Sellers"
- "New Arrivals"
- Category-specific collections
- Add to navigation menu

**Step 9: Design Homepage**

Create an engaging homepage:
- Hero section with clear value proposition
- Featured products section
- Benefits/trust signals
- About section
- Email signup form
- Footer with links

**Step 10: Mobile Optimization**

- Preview on mobile device
- Check all pages load properly
- Test navigation and checkout
- Ensure images display correctly
- Verify buttons are clickable

**Step 11: Final Checks**

Before launching:
- [ ] Test checkout process end-to-end
- [ ] Verify all links work
- [ ] Check spelling and grammar
- [ ] Ensure contact information is visible
- [ ] Test on different devices
- [ ] Place test order

**Step 12: Go Live**

- Remove password protection
- Announce your launch
- Share on social media
- Start driving traffic

## Key Takeaways
- Complete all setup steps systematically before launching
- Customize theme to match your brand and niche
- Install essential apps early for functionality
- Create legal pages to protect your business
- Test everything thoroughly before going live
